Puzzle Time

＃企劃發想＃流程規劃＃介面設計＃Pototype

DESCRIPTION

發想來自在工作的時候，時常無法將工作天數評估良好，造成同一時間的工作量暴增，甚至是接回來才發現其他部門的工作量早已不堪負荷。

已現有的行事曆軟體都是以自己算時間，然後寫上事項，但其實很花時間；如果這時候，新增事項是可以用工作天數為估算，並把每一個事項的工作天數填完後再新增，這樣子所有案子的時程就一目瞭然，而且再新增事項上所花的時間也會少很多。

艾蜜莉定存股App

＃介面設計＃Pototype＃使用者研究＃Ios/Android雙平台

DESCRIPTION

艾蜜莉定存股，是依據小資女艾蜜莉的投資策略，所規劃出的軟體。在之前只有桌機版本，後來依據App的使用情境與體驗，在2017年推出App版本。剛推時出時在Ios的熱門搜尋上出現，並且登上Ios不分類免費下載第13名。

目前上架一年，累積下載量已超過40萬，每週活躍用戶達3萬人。

POSITION

當初在使用者族群上，除了鎖定既有正在使用桌機版軟體的人之外，希望吸引小資族群。所以在顏色上使用與小資女形象相符的粉紅色，並且以白色作為底色。營造出有別於股市黑底的形象。

PROCESS

除了擔任視覺設計之外，因為要與工程師溝通，所以也需要準備各式的文件，包含UI Flow。
